核心特性

为创作者精心设计的每一个细节

零构建架构

无需 npm install、无需打包工具、无需 node_modules。双击 HTML 即可运行，五年后依然有效。

国内优先

OSS 图片、网易云、B站视频等国内可访问组件作为默认选项，海外组件作为可选增强。

组件化设计

图片、音频、视频、文章等媒体组件标准化，复制粘贴即可使用，开箱即用。

设计系统

Tailwind CSS + 标准动效库 + 字体系统，统一的视觉语言，专业的呈现效果。

安全可靠

CSP 安全策略、HSTS、X-Frame-Options 等安全头配置完善，防御 XSS 攻击。

本地化依赖

核心依赖下载到 vendor/，CDN 仅作备用。离线环境也能正常工作。

分层架构

清晰的职责划分，按需引入

第四层 自动化层

CLI 工具、CDN 映射、缓存管理

第三层 内容层

图片、音频、视频、文章配置

第二层 特色功能层

聚光灯效果、交互画廊、纵向滚动

第一层 基础层

零构建模板、设计系统、媒体组件、工具函数

git clone git@github.com:MiLab-Bit/MCreator.git
2

编辑配置

修改 site.config.json 填写站点信息

{
  "site": {
    "name": "我的作品集",
    "url": "https://mysite.com"
  },
  "storage": {
    "primary": { "provider": "aliyun-oss" }
  }
}
